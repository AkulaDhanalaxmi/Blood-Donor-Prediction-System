
# Blood Donor Prediction System

## Overview

This project uses Machine Learning techniques to predict blood donor availability and eligibility based on donor information and health-related attributes. The goal is to improve donor management and support efficient blood bank operations.

## Features

* Data preprocessing and cleaning
* Handling missing values and outliers
* Feature engineering and label encoding
* Class imbalance handling using SMOTE
* Dimensionality reduction using PCA
* Model training and evaluation
* Performance comparison of multiple machine learning algorithms

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* XGBoost

## Machine Learning Models

* Logistic Regression
* Support Vector Machine (SVM)
* K-Nearest Neighbors (KNN)
* Decision Tree
* Random Forest
* AdaBoost
* Gradient Boosting
* XGBoost

## Dataset

* Blood Donor Dataset
* Includes donor demographic and health-related information

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-Score
* ROC-AUC Score

## Results

Multiple classification algorithms were evaluated and compared to identify the best-performing model for blood donor prediction.
